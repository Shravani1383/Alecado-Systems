# 3D Image To Floor Plan Generator
## Tkinter Application to convert your 3D designs to 2D 
## Getting Started
```
git clone https://github.com/Shravani1383/Alecado-Systems.git
```
```
pip install opencv-python
```
```
pip install tkinter
```
```
pip install matplotlib
```
